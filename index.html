核心功能

为什么选择 OpenClaw？

专为开发者和高级用户设计的 AI 网关，让你随时随地与 AI 助手对话

自托管

运行在你自己的硬件上，完全掌控你的数据和隐私。无需依赖第三方云服务。

多渠道

一个网关同时服务 WhatsApp、Telegram、Discord、iMessage，统一入口管理所有对话。

Agent 原生

专为编码助手设计，支持工具调用、会话管理、记忆功能和多 Agent 路由。

开源

MIT 协议开源，社区驱动。自由定制、扩展和贡献代码。

移动节点

支持 iOS 和 Android 节点配对，实现 Canvas、相机、屏幕录制、位置等丰富功能。

安全可靠

支持令牌认证、白名单、提及规则等安全控制，确保只有授权用户可以访问。

核心优势

OpenClaw 的独特之处

极速部署

5 分钟即可完成安装和配置。Node.js 22+ 环境，一个 npm 命令搞定。

多 Agent 路由

隔离的会话管理，每个工作区或发送者拥有独立的 Agent 会话，互不干扰。

富媒体支持

支持图片、音频、文档的发送和接收。可选语音转文字钩子。

Web 控制面板

浏览器仪表板管理聊天、配置、会话和节点。支持本地和远程访问。

工具流式传输

支持 Agent 工具调用的实时流式传输，长响应自动分块处理。

群组聊天支持

基于提及的群组激活机制，支持 @openclaw 触发响应。
